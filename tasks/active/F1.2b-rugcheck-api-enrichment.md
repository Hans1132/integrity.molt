# F1.2b — RugCheck API enrichment (nezávislé na F1.2a)

## Kontext
RugCheck API (api.rugcheck.xyz) je veřejné a volně dostupné.
Poskytuje risk score pro libovolný Solana token mint.
Může nahradit/doplnit SolRugDetector dataset dokud není dostupný.

## Co udělat
1. Přidat RugCheck lookup do scan pipeline:
   - `src/scanners/rugcheck.js` — wrapper pro `GET https://api.rugcheck.xyz/v1/tokens/{mint}/report/summary`
   - Volat při `scan/quick` a `scan/token`
2. Výsledek zahrnout do scan reportu jako `rugcheck` sekci:
   ```json
   {
     "rugcheck": {
       "score": 850,
       "risks": ["mint_authority_enabled", "low_liquidity"],
       "verdict": "warn"
     }
   }
   ```
3. Pokud mint je v `known_scams` DB → skip API call, použít DB hodnotu

## Acceptance criteria
- [ ] RugCheck data v scan reportu
- [ ] Fallback při API timeout (max 3s)
- [ ] `bash scripts/test-gate.sh` PASS

## Soubory ke změně
- `src/scanners/rugcheck.js` (nový)
- `src/scanners/token.js` (přidat volání)
- `src/scanners/quick.js` (přidat volání)

## Poznámky
- API je free, bez auth
- Rate limit: ~10 req/s
- Docs: https://api.rugcheck.xyz
