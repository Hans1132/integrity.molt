# F1.2a — Import solrpds CSV datasetu

## Změna zdroje (2026-04-15)
Původní plán (SolRugDetector z arxiv paperu 2603.24625) **zrušen** — GitHub repo
autorů není veřejně dostupný a email kontakt zůstal bez odpovědi. Hans rozhodl

## Zdroj dat
Soubor: `data/scam-datasets/solrpds_2021..2024.csv`
Pokrytí: rug pull tokeny Solana 2021–2024.

## CSV Schema (zdokumentováno 2026-04-15)

Soubory: `data/scam-datasets/solrpds_{2021,2022,2023,2024}.csv`
Celkem řádků: ~116 308 (včetně hlaviček), ~116 304 datových řádků

| Sloupec | Typ | Popis |
|---|---|---|
| LIQUIDITY_POOL_ADDRESS | text (base58) | Adresa liquidity pool účtu (Raydium/Orca) |
| MINT | text (base58) | **Klíčové pole** — mint adresa tokenu (importujeme do known_scams) |
| TOTAL_ADDED_LIQUIDITY | float | Celková přidaná likvidita (v SOL nebo LP tokenech) |
| TOTAL_REMOVED_LIQUIDITY | float | Celková odebraná likvidita |
| NUM_LIQUIDITY_ADDS | integer | Počet přidání likvidity |
| NUM_LIQUIDITY_REMOVES | integer | Počet odebrání likvidity |
| ADD_TO_REMOVE_RATIO | float | Poměr přidání/odebrání (< 1.0 = rug indikátor) |
| LAST_POOL_ACTIVITY_TIMESTAMP | datetime | Poslední aktivita v poolu (UTC) |
| FIRST_POOL_ACTIVITY_TIMESTAMP | datetime | **Mapováno na first_seen_at** — první aktivita v poolu (UTC) |
| LAST_SWAP_TIMESTAMP | datetime | Poslední swap v poolu |
| LAST_SWAP_TX_ID | text (base58) | TX sig posledního swapu |
| INACTIVITY_STATUS | text | **Klíčové pole** — `Inactive` = potvrzený rug pull, `Active` = aktivní (podezřelý) |

### Mapování na known_scams schema

| CSV pole | known_scams sloupec | Poznámka |
|---|---|---|
| MINT | mint | Primární klíč; validace délky 32-44 znaků |
| INACTIVITY_STATUS = "Inactive" | rug_pattern = 'inactive_pool' | confidence = 0.90 |
| REMOVED > ADDED × 1.2 | rug_pattern = 'liquidity_drain' | confidence = 0.50 |
| jinak | rug_pattern = 'active_suspicious' | confidence = 0.50 |
| FIRST_POOL_ACTIVITY_TIMESTAMP | first_seen_at | ISO 8601 UTC normalizace |
| (není v CSV) | creator | NULL — SolRPDS neobsahuje creator wallet |
