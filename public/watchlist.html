Alert types

What triggers an alert

Helius webhooks deliver real-time on-chain events. Every alert is signed.

🔑
Authority Changes
Detect when mint authority, freeze authority, or program upgrade authority is transferred or revoked. The earliest possible rug signal.
📦
Large Transfers
Alert on large SOL or token transfers out of monitored addresses. Configurable thresholds per tier.
⚡
Program Upgrades
Know immediately when an on-chain program is upgraded. Upgradeable programs are a critical risk factor.
📊
Risk Escalations
Periodic risk rescans detect when a low-risk address becomes high-risk. Notifies when risk level changes.
📝
Delta Reports
Weekly (Pro Trader) or daily (Builder/Team) signed diff reports showing exactly what changed since the last scan.
🎯
Custom Alert Rules
Team tier: define custom conditions for alerts — specific programs, transfer amounts, or address patterns.

Setup

Start monitoring in 30 seconds

Use the Telegram bot or the API — no installation required.

1

Open the Telegram bot

Start a conversation with @intmolt_bot on Telegram.

/start
2

Add an address to your watchlist

Use /watch followed by any Solana address. You can add a label to identify it.

/watch EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v USDC
3

Receive real-time alerts

You'll get a Telegram message the moment something changes on-chain. Every alert includes the risk level, what changed, and a link to the full signed report.

4

Upgrade for more coverage

Use /upgrade in Telegram to see subscription options and pay with USDC or card.

/upgrade

API: add to watchlist programmatically

POST /watchlist/add
X-API-Key: im_your_builder_key

{
  "address": "EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v",
  "label": "USDC Mint",
  "notify_email": "alerts@yourteam.com"
}
