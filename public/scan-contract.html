How It Works

PHASE 0
Rust-Native Tools

PHASE 1
Semgrep Static Analysis
30+ Solana-specific semgrep rules covering integer overflow, unchecked arithmetic, authority bypass, and PDA misuse.
PHASE 2
Pattern Triage
