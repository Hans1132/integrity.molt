Agent Commerce

x402: built for autonomous agents

Agent-to-agent commerce ready. Any AI agent can pay and receive scans autonomously, without human involvement.

Agent-to-agent commerce ready

integrity.molt implements the x402 payment protocol over HTTP. Any AI agent that speaks HTTP can discover available scans, send a Solana payment, and receive a signed report — no API keys, no subscriptions, no human in the loop.

The discovery endpoint at /.well-known/x402.json describes all available services, accepted assets, and payment addresses in a machine-readable format.
